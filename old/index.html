Code and Data
imagetrans code. Code package written using Matlab for doing generic object recognition using statistical models. Under the GNU General Public License. Read the readme files in the docs directory before using the code.
robomedia data. Data we used for the HLT-NAACL workshop on Learning Word Meaning from Non-Linguistic Data, 2003.
Corel data. Data we used for the unpublished manuscript (Carbonetto and de Freitas, 2003).
feature weighting code. EM for a mixture of Gaussians with Bayesian shrinkage priors for feature weighting. The Matlab code was used for the AI-Stats 2003 paper.
gibbs sampling code. A small Matlab script which demonstrates Gibbs sampling moves on a bivariate Gaussian.
blocked gibbs sampler code. This script hows that sampling on the marginal instead of the conditional can significantly increase convergence, even on a simple example.

Masters Projects
Robust Object Detection Using Boosted Learning. Machine Learning term project. A face detection algorithm using the AdaBoost algorithm, as motivated by Viola and Jones. The training data is now available.

An Implementation for multiple dispatch in Java using the Elide framework.
Topics in Programming Languages term project. No. 11 in Google for "multiple dispatch Java". Click here to obtain the software. The software was developed using the ELIDE framework. Click here to see my presentation on multiple dispatch.

Playing Pictionary: an exploration in human-computer interaction
Human-Computer Interaction term project. A study to determine whether methods exist to mitigate the effects of distance on collaborative communication tasks in computer-mediated, as opposed to face-to-face, conditions.

Implementing an Improved Smooth Navigation Controller I designed a navigation module for José the Robot that is both both efficient and able to negotiate obstacles.

Using Dynamic Decision Networks to Make Rational Tutorial Actions
I use dynamic decision networks for the purpose of modeling the actions of a logic gate tutoring system. Click here to download the undocumented Visual C++ for Windows project. It uses the FOX graphical user interface toolkit and the SMILE portable inference engine for decision-theoretic models.

Modeling novelty processes in the perirhinal cortex
For Computational Neuroscience course. I show the results of preliminary investigations on modeling novelty, recency and familiarity in the perirhinal cortex.

Presentation on visual attention.
For Human-Computer Interaction course. I examine motivation for understanding visual attention mechanisms for HCI.

Presentation on Adaline and gradient descent.
Introductory presentation on neural networks for the Computational Neuroscience course.
