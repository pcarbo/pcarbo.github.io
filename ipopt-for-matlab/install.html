MATLAB interface for IPOPT

About      Download      Installation      Tutorial

To install IPOPT for MATLAB, simply unzip or uncompress the file in a directory, start up MATLAB in that same directory (or make sure the location of the files are in your MATLAB path) and type "help ipopt" in the MATLAB prompt to get started.

Troubleshooting. If you have problems using the MATLAB interface, or have possibly encountered a bug in the code, please consult the archives of the IPOPT mailing list. If you can't find the answer there, try sending an email to the IPOPT mailing list.
