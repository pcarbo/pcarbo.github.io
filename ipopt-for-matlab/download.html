File name Description
ipopt-linux.tar.gz (1.6 MB) For MATLAB versions 7.4 through 7.6 on the Linux operating system and using an Intel processor, built with gcc 4.1.1, IPOPT 3.5.3 and MUMPS indefinite linear solver 4.8.1.
ipopt-intelmac.tar.gz (1.5 MB) For MATLAB versions 7.4 through 7.6 on Mac OS X and using an Intel processor, built with gcc 4.0.1 and gfortran 4.3.0, IPOPT 3.5.4 and MUMPS indefinite linear solver 4.8.1. (Note: To get this to work, you may need to set the DYLD_LIBRARY_PATH environment variable to point to the location of the gfortran libraries installed on your system.)

If none of the software packages above fit the description of your system setup, you may have to build the MATLAB interface from the source. If this is the case, you will need to consult the IPOPT project webpage. You may also request a package compiled under different settings by contacting the author or sending an email to the IPOPT mailing list.
